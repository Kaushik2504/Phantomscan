# 👻 PhantomScan

A custom all-in-one reconnaissance tool built in Python for basic security testing and information gathering.

---

## 🚀 Features

- 🔍 Subdomain Enumeration
- 🌐 IP Address Resolution
- 🔓 Port Scanning (File-based with service names)
- ⚙️ Technology Detection (Headers + Content-based)
- 📂 Directory Bruteforce
- 📄 Report Generation

---

## 🛠️ Requirements

- Python 3.x
- requests module

Install dependencies:
pip install requests

---

## 📦 Setup

1. Clone the repository:
git clone https://github.com/Kaushik2504/phantomscan.git

2. Navigate to the folder:
cd phantomscan

3. Add required files:
- ports.txt (list of ports with service names)
- dirs.txt (directory wordlist)

---

## ▶️ Usage

Run the tool:
python all_in_one_tool11.py

Enter target:
example.com
